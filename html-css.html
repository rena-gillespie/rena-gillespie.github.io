The Best Practices Associated with using Classes vs. Ids

When wanting to customise a blog, theres many ways to assign colour, size and/or font type

and there's many ways to do that, two of them is using "ID" and "class".

Class

Using "Class" can assign styles multiple times to elements for example, texts.
This is useful if your blog has colour schemes to use on many elements.
You can use this by typing ".insert-name" and add a class="insert-name" to your elememnt

ID

Assigning IDs is used for one element
This comes in handy when you want to give unique atribute to something
Maybe give a sentence a colour that deviates from the colour scheme!
